Code changes (precision)
------------------------
  - `junction_path` transcripts use per-path **bottleneck** node coverage/longcov and run through
    the **full predcluster** chain (pairwise, isofrac, readthr, dedup). Ref/trace assists
    (`ref_chain*`, `junction_chain`, …) stay on the legacy bypass unless `RUSTLE_PROTECT_JUNCTION_PATH`
    is set (restores old junction_path bypass for debugging).
  - Paths weaker than **readthr** on both coverage and longcov are not emitted from DFS.
  - Default **max_junction_paths** lowered 10000 → **1200** per bundle; `--max-junction-paths`
    overrides. `--max-sensitivity` still enables junction-path enumeration with the new cap.
  - Default **-E / junction_correction_window** 25 → **30** bp; default **RUSTLE_EMIT_REF_SSDIST**
    40 → **48** for `find_guide_pat` / ref assist snapping (helps gffcompare **j**-class near misses).
